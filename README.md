## 项目介绍

这是一个基于HTML5、CSS3和JavaScript的现代化2048游戏实现。游戏设计精美，响应式布局，支持多种设备，并提供了丰富的自定义选项和增强功能。

## 主要功能

- 🎮 **多种游戏模式**: 标准模式和无尽模式
- 📐 **可变网格大小**: 支持3×3到10×10的自定义网格
- 🎯 **难度级别**: 简单、中等、困难三种难度
- 🤖 **AI游戏分析**: 实时分析最佳移动方向和游戏状态
- 🎨 **主题定制**: 支持浅色、深色、跟随浏览器以及完全自定义主题
- 💾 **数据管理**: 游戏进度保存、导入/导出功能
- 📱 **响应式设计**: 完美适配桌面和移动设备
- ⚡ **性能优化**: GPU加速、动画控制等选项
- 🔧 **丰富设置**: 多种可配置选项提升游戏体验

## 游戏玩法

### 基本规则
- 使用 **方向键** 或 **WASD** 键移动方块
- 相同数字的方块碰撞时会合并，数字相加
- 每次移动后，会在空白位置生成新的方块（2或4）
- 标准模式：达到2048获胜
- 无尽模式：尽可能获得高分

### 触摸控制
- 在移动设备上支持触摸滑动操作
- 向左/右/上/下滑动屏幕移动方块

## 功能详情

### 游戏模式
- **标准模式**: 目标达到2048，达成后可继续挑战更高分数
- **无尽模式**: 没有获胜目标，尽可能获得更高分数

### 网格大小
- **预设尺寸**: 3×3、4×4、5×5
- **自定义尺寸**: 支持3×3到10×10的任意矩形网格
- **正方形锁定**: 可选择保持网格为正方形

### 难度级别
- **简单**: 4出现的概率40%，得分乘数1
- **中等**: 4出现的概率10%，得分乘数1（默认）
- **困难**: 4出现的概率5%，得分乘数1.5

### AI游戏分析
- **最佳移动方向**: 分析当前棋盘，推荐最佳移动方向
- **合并机会评估**: 识别潜在的合并机会
- **游戏状态评估**: 评估当前游戏状态（良好/一般/危险）
- **得分潜力分析**: 预测未来得分潜力
- **支持网格**: 3×3、4×4、5×5标准网格

### 自定义主题
- **预设主题**: 浅色、深色、跟随浏览器
- **完全自定义**: 
  - 基础颜色（背景、主色调、次色调等）
  - 滑块颜色（每个数字对应的背景色和文字颜色）
  - 模态框颜色
- **实时预览**: 自定义时可实时预览效果
- **保存/加载**: 自定义主题可保存到本地

### 数据管理
- **自动保存**: 游戏进度自动保存到本地存储
- **数据导出**: 将当前游戏状态导出为Base64编码文本
- **数据导入**: 从Base64编码文本导入游戏状态
- **数据复制**: 一键复制导出数据到剪贴板
- **数据清除**: 清除所有本地数据

### 设置选项
- **游戏性能**:
  - GPU加速（提升动画性能）
  - 滑块动画开关
- **外观设置**:
  - 主题选择（自动/浅色/深色/自定义）
- **数据管理**:
  - 导入/导出游戏数据
  - 清除所有数据

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| **方向键** | 移动方块 |
| **WASD** | 移动方块 |
| **P / ESC** | 暂停/继续游戏 |
| **Z (Ctrl+Z)** | 撤销上一步 |
| **R** | 重新开始游戏 |
| **N** | 切换到标准模式 |
| **E** | 切换到无尽模式 |
| **G** | 循环切换网格大小 |
| **L** | 打开AI分析面板 |

## 技术特性

- **前端技术**: HTML5、CSS3、原生JavaScript
- **样式框架**: Tailwind CSS
- **图标库**: Font Awesome 6
- **响应式设计**: 媒体查询和Clamp函数
- **本地存储**: 使用LocalStorage保存游戏状态和设置
- **动画效果**: CSS transitions和keyframes
- **GPU加速**: 使用transformZ提升性能
- **无障碍**: ARIA标签和键盘导航支持
